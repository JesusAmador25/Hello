## Creación de repositorios
En la parte superior derecha de la interfaz de Github hay un icono estilo libro para crear un nuevo repositorio.
Añadimos titulo y descripcion breve y aceptamos.

## Solicitud de pull request
- Primero realizamos un fork al repositorio al que haremos pull request
- Posteriormente solicitamos el pull desde el repositorio al que hicimos fork
- Añadimos mensaje de la causa de la solicitud
- Ser pacientes para que el dueño del repositorio original acepte el pull request

## Posibles conflictos al hacer un merge
- Conflictos de linea (sobreescritura en la misma linea)
